# Library Management

- basic library management system
- kotlin with gradle(preferred) or java
- any framework (spring preferred)
- mongodb
- web apis only
- proper commits and make sure to tag each step

## requirements

- users, signup and login
- admin
- admins can add books
- users can borrow books
- each book can be borrowed once
- users can list books they have borrowed
- users can return books
- some books have expiry, after some time they are automatically returned back to library
- some books have policy, they will return to library at end of the day(10pm), mimicking only read at library.
